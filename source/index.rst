Welcome to IDC hackpack!
=========================================

Welcome to the Intel® Tiber™ Developer Cloud (IDC) Hackpack! This guide is designed to help hackers and developers quickly get started with building Generative AI applications using Intel's production cloud for AI.


.. image:: ./images/hackpack.png
   :alt: hackpack fun

| 

As the compute sponsor for the hackathon, Intel provides access to powerful resources on IDC, including Intel® Data Center GPU Max Series 1100 GPUs and CPU VMs. This hackpack serves as a reference to help you navigate the IDC environment, develop your AI applications efficiently, and deploy them for showcasing.
